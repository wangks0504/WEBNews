# WEB新闻发布系统 - 开发流程图

## 🏗️ 整体开发流程

```mermaid
flowchart TD
    Start([开始项目])

    Setup[搭建项目架构 + 配置JWT]
    Meeting1[任务分配会议]

    Dev[各自开发功能模块]
    Problem[遇到问题 -> 先自查 → AI → 群里问 → 协调解决]

    SelfTest[本地自测 + Swagger验证]
    Commit[提交代码到各自分支]

    Merge[合并代码]
    Integration[联调测试]
    Fix[修Bug]

    Report[撰写项目报告]
    End([项目完成])

    Start --> Setup --> Meeting1 --> Dev --> Problem --> SelfTest --> Commit --> Merge --> Integration --> Fix --> Report --> End

```

## 🔄 Git协作流程

```mermaid
flowchart TD
    Start([开始开发])
    
    Clone[克隆项目到本地]
    Branch[创建自己的分支]
    Code[编写代码]
    
    Status[git status 查看修改]
    Add[git add . 添加修改]
    Commit[git commit -m 提交]
    Push[git push 推送到远程]
    
    Wait[等待王世强合并]
    Merge[王世强合并到主分支]
    Pull[git pull 拉取最新代码]
    
    Conflict{有冲突?}
    Resolve[解决冲突]
    
    Done([完成])
    
    Start --> Clone
    Clone --> Branch
    Branch --> Code
    Code --> Status
    Status --> Add
    Add --> Commit
    Commit --> Push
    Push --> Wait
    Wait --> Merge
    Merge --> Pull
    Pull --> Conflict
    
    Conflict -->|否| Done
    Conflict -->|是| Resolve
    Resolve --> Add
```
