## 📚 给小白的快速入门指南

### 第1步：搭建开发环境（第1天上午）
```bash
# 1. 安装 Visual Studio 2022 或 VS Code
# 2. 安装 .NET 6/7 SDK
# 3. 安装 SQL Server 2019

# 4. 克隆项目
git clone [项目地址]
cd WEBNews

# 5. 恢复依赖
dotnet restore

# 6. 运行项目
dotnet run --project WEBNews.API
```

### 第2步：理解项目结构（第1天下午）
```
WEBNews/
├── WEBNews.API/          # 你要写的控制器在这里
├── WEBNews.Models/       # 你要写的数据模型在这里
├── WEBNews.Data/         # 数据库相关（组长和高手已配置好）
└── WEBNews.Services/     # 你要写的业务逻辑在这里
```

### 第3步：创建你的第一个接口（第2天）
**示例代码（供参考）：**

```csharp
// 1. 创建控制器
[ApiController]
[Route("api/[controller]")]
```
