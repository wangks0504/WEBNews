## 🗂️ 数据库表设计（王世强负责）

### 1. Users（用户表）
```sql
- UserId (主键)
- UserName (用户名)
- PasswordHash (密码哈希)
- Email (邮箱)
- RealName (真实姓名)
- CreatedAt (创建时间)
```

### 2. News（新闻表）
```sql
- NewsId (主键)
- Title (标题)
- Content (内容)
- Summary (摘要)
- AuthorId (作者ID，外键)
- PublishTime (发布时间)
```

### 3. Attachments（附件表）
```sql
- AttachmentId (主键)
- FileName (文件名)
- FilePath (文件路径)
- FileSize (文件大小)
- FileType (文件类型)
- NewsId (新闻ID，外键)
- UploadTime (上传时间)
```

### 4. NewsReceivers（新闻接收人表）
```sql
- Id (主键)
- NewsId (新闻ID，外键)
- ReceiverId (接收人ID，外键)
- AssignedAt (指定时间)
```
